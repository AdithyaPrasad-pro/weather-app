#  Weather App

A simple and modern Weather App that fetches real-time weather data using an API.
Built using HTML, CSS, and JavaScript.

---

#  Live Demo

👉 https://adithyaprasad-pro.github.io/weather-app/

---

# Features

* Search weather by city name
* Displays temperature in Celsius
* Shows weather condition (e.g., Clouds, Rain, Clear)
* Clean and responsive UI

---

# Tech Stack

* HTML
* CSS
* JavaScript
* OpenWeatherMap API

---

# Project Structure

weather-app/

---

# How It Works

1. User enters a city name
2. App sends request to weather API
3. Receives JSON data
4. Displays temperature, condition, and icon
