# Weather App

A modern weather application that fetches real-time weather data using an API. Built using HTML, CSS, and JavaScript with a clean UI and dynamic styling.

## Live Demo

https://adithyaprasad-pro.github.io/weather-app/

## Features

* Search weather by city name
* Displays temperature in Celsius
* Shows weather condition
* Displays humidity and wind speed
* Dynamic weather icons
* Background changes based on weather conditions
* Loading state while fetching data
* Handles invalid city input gracefully
* Press Enter to search
* Smooth UI animations

## Tech Stack

* HTML
* CSS
* JavaScript
* OpenWeatherMap API

## Project Structure

```
weather-app/
├── index.html
├── style.css
└── script.js
```

## How It Works

1. User enters a city name
2. Application sends a request to the weather API
3. Receives JSON data
4. Displays temperature, condition, humidity, wind speed, and icon
5. Updates UI dynamically based on weather conditions
